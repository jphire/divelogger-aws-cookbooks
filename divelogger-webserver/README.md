# divelogger-cookbook

This cookbook sets up an express web server for DiveLogger.

## Supported Platforms

ubuntu-12.04, centos-6.5

## Attributes

## Usage

### divelogger::default

Include `divelogger` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[divelogger::default]"
  ]
}
```
